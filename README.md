# Аэродиск
## Тестовое задание

Веб-приложение на **FastAPI** для работы с дисками в среде Linux

Позволяет:
1. Просматривать список доступных дисков
2. Монтировать диски в выбранную точку
3. Демонтировать
4. Форматировать в файловые системы ext4, xfs, btrfs
